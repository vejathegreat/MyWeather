# MyWeather

***Api Details***
- Open Weather Api
- https://openweathermap.org/

***Language & Platform***
- Kotlin
- Android

***Libraries***
- RxJava2
- Glide
- Retrofit
- SimpleLocation
- lottie
- Jetpack Compose 
- Androidx

***Device Requirements***
- Minimum api level: SDK 16 (Jelly Bean)
- Network connection
- Location Permission

***Setup***
- Insert you api in local.properties ( apiKey="YOUR API KEY")
